# Data Analyser — Frontend

React 19 + TypeScript + Vite single-page app for the Data Analyser project.

➡️ **See the [root README](../README.md)** for the full project overview, architecture, and setup instructions.

## Quick start

```bash
npm install
cp .env.example .env   # set VITE_API_URL (defaults to http://localhost:5000/api)
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start the Vite dev server |
| `npm run build` | Type-check (`tsc -b`) and build for production |
| `npm run preview` | Preview the production build locally |
| `npm run lint` | Run ESLint |
